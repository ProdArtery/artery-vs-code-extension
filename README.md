<h1 align="center">Your APIs. Handled For You.</h1>

<p align="center">
  Artery is an AI-powered API workflow tool built for macOS — with Docker, snapshots, and more.<br/>
  This extension connects your VS Code AI agent directly to Artery, so it can create collections, send requests, and read responses — all without leaving your editor.
</p>

<p align="center">
  <strong>No files get configured. Just install and go.</strong>
</p>

---

## Setup

**1. Open the Artery app**

**2. Turn on the AI server**

Click the MCP icon in the Artery toolbar. When it turns **green**, the AI server is on and ready.

<p align="center">
  <img src="ai-server-on.png" width="320" alt="Artery AI server turned on — green icon" />
</p>

> Red means off. Green means on.

**3. Install this extension in VS Code**

That's it, your AI agent now has full access to your Artery workspace.

---

## What your agent can do

| Tool | Description |
|---|---|
| **List Collections** | View all collections and folders |
| **Read Collection** | See all requests inside a collection |
| **Create Collection** | Create new collections, including nested ones |
| **Create Request** | Create an API request (GET, POST, etc.) with URL, body, and content type |
| **Send Request** | Execute a stored request and return the response |
| **Read Response** | Read the last response — status, body, headers, timing |
| **Modify Request** | Update URL, method, body, or content type on an existing request |

> **Note:** Auth credentials must be set manually in Artery's Auth tab.
